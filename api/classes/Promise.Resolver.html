Promise.Resolver Class

Defined in: src/parcela.github.io/assets/core.js:2654
Module: event-emitter
Parent Module: event

Represents an asynchronous operation. Provides a standard API for subscribing to the moment that the operation completes either successfully (fulfill()) or unsuccessfully (reject()).

Provided by the Parcela module.

Defined in src/parcela.github.io/assets/core.js:2812

If value is a promise or a thenable, it will be unwrapped by recursively calling its then method. If not, the resolver will be fulfilled with value.

This method is called when the promise's then method is called and not in resolve to allow for lazy promises to be accepted and not resolved immediately.

Parameters:

  • value Any

    A promise, thenable or regular value

Resolves the promise, signaling successful completion of the represented operation. All "onFulfilled" subscriptions are executed and passed the value provided to this method. After calling fulfill(), reject() and notify() are disabled.
