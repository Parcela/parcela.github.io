Parcela is a open-source project managed by the community-reviewers. Not only do you have access to the sourcecode, but you can file issues, make pull request or create components for your own usage.

Contribute can be done in 3 ways:

1. File issues
Everyone can file issues. Whenever you see a bug, are think Parcela's framework lacks some feature, just file an issue here.

2. Contribute to the code*

3. Become a commiter*

4. Become a reviewer*

* Be sure to sign a [CLA](http://yuilibrary.com/contribute/cla/) so your changes can be pulled in (see below).
